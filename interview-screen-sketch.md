# Skiss: Förbered intervju-skärm

```
+---------------------------------------------------+
|                 Förbered intervju                 |
+---------------------------------------------------+
| [Dropdown]  Välj språk: [ Svenska  v ]            |
| [Chips]     Svårighetsgrad: [ Lätt  Medel  Svår ] |
| [Chips]     Kategori: [ Algoritm  System  Beteende]
| [Chips]     Riktning: [ Frontend  Backend  Java ] |
| [Chips]     Format: [ Video  Text  Quiz ]         |
| [Slider]    Tidslängd:  [====|-----] 30 min       |
| [Stepper]   Antal frågor:  [ - ] 5 [ + ]          |
| [Button]    Testa kamera/mikrofon                 |
| [Button]    Visa exempelfråga                     |
+---------------------------------------------------+
| [Button]    Starta intervju                       |
+---------------------------------------------------+
| (Ev. preview/video-thumbnail om format=video)      |
+---------------------------------------------------+
```

## Komponenter:

- **Rubrik:** "Förbered intervju"
- **Dropdown:** För språkval
- **Chips:** För svårighetsgrad, kategori, riktning, format
- **Slider:** För tidslängd
- **Stepper:** För antal frågor
- **Button:** Testa kamera/mikrofon
- **Button:** Visa exempelfråga
- **Button:** Starta intervju (primär CTA)
- **(Ev. preview/video-thumbnail om video är valt)**

---

Denna skiss kan användas som grund för att bygga UI-komponenterna steg för steg.
