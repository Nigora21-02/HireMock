# Flödeschema: Starta intervju

## 1. Hemskärm

- Knapp: "Starta intervju"
- Navigerar till "Förbered intervju"

## 2. Förbered intervju-skärm

Här väljer användaren:

- Språk (svenska, engelska)
- Svårighetsgrad (lätt, medel, svår)
- Kategori (Algoritmer, Systemdesign, Beteende)
- Riktning/roll (Frontend, Backend, Java, DevOps)
- Format (Video, Text, Quiz)
- Tidslängd (15, 30, 60 min)
- Antal frågor
- Valfritt: Testa kamera/mikrofon
- Valfritt: Visa exempel på frågor

**UI-förslag:**

- Chips/selectors för kategori, roll, svårighet
- Dropdown/modal för språk
- Slider/stepper för tid/antal frågor
- Preview-ikon eller video-thumbnail för videoformat

## 3. Starta intervju

- Tryck på "Starta"
  - Video: nedräkning (3,2,1), starta kamera
  - Text/quiz: visa första frågan
- Visa: "Intervjun pågår", timer, möjlighet att pausa/avsluta

## 4. Under intervjun

- Visa fråga, svarsfält (eller video-inspelning), timer
- Progressbar för antal frågor
- Möjlighet att hoppa över fråga

## 5. Efter intervjun

- Visa resultat, feedback, möjlighet att spara/dela

---

## Sammanfattning av valbara steg/parametrar

- Språk
- Svårighetsgrad
- Kategori/ämne
- Riktning/roll
- Format (video/text/quiz)
- Tidslängd/antal frågor
- Testa teknik (kamera/mikrofon)
- Exempelfråga/preview

---

Detta schema kan användas som grund för att bygga och checka av varje steg i projektet.
